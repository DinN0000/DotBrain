# DotBrain

AI-powered PKM (Personal Knowledge Management) organizer for macOS.

DotBrain lives in your menu bar and automatically organizes your notes using the PARA method (Projects, Areas, Resources, Archive).

## Install

```bash
npx dotbrain
```

Requires macOS 13+ and Node.js 18+.

## What it does

- Watches your `_Inbox/` folder for new notes
- AI classifies and moves them to the right PARA folder
- Generates semantic links between related notes
- Builds AI companion files for each folder

## Uninstall

```bash
npx dotbrain --uninstall
```
